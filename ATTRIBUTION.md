# Attribution

Handshake Wallet (formerly Bob Wallet) is built upon the foundation of several open-source projects. We would like to express our gratitude to the individuals and organizations who have contributed to the software that makes this project possible.

## Original Codebase
**Bob Wallet**
* Originally created and maintained by **Kyokan** and the Bob Wallet Contributors (2018-2025) upto version 2.1.0.
* Handshake Wallet is a fork of Bob Wallet, and much of the core infrastructure and UI patterns were established by the original development team.
* [Bob Wallet on GitHub](https://github.com/kyokan/bob-wallet)

## Handshake Protocol
**Handshake Daemon (hsd)**
* Handshake Wallet integrates a full Handshake node directly into the application. This is powered by `hsd`, the Handshake daemon written in Node.js.
* Copyright (c) 2017-2020, Christopher Jeffrey (MIT License).
* [hsd on GitHub](https://github.com/handshake-org/hsd)

## Hardware Wallets
**Ledger Support**
* Ledger hardware wallet integration is made possible through libraries and open-source contributions provided by Ledger.
* [LedgerHQ on GitHub](https://github.com/LedgerHQ)

## Handshake Wallet Modifications
* Additional features, rebranding, multi-select domain management, and ongoing development from 2026 onward are authored by **Christopher Istvan / Webelity**.
