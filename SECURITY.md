# Security Policy

## Supported Versions

Please see [Releases](https://github.com/webelity/hns-wallet/releases).  
We recommend using the [most recently released version](https://github.com/webelity/hns-wallet/releases/latest).

## Reporting a Vulnerability

Please don't report security issues on GitHub. Instead, send an e-mail to security@webelity.com describing your issue.
